Conjunction Junction Function

I had the Conjunction Junction song stuck in my head and thought it'd be fun to make a function that outputs a random lyric from it.

Named Function

            
            function(parameter){
                Do something();
            }
            

            function randomLyrics(lyrics) {
                const randomIndex = Math.floor(Math.random() * lyrics.length);
                return lyrics[randomIndex];
            }

            const songLyrics = [
                "Hooking up words and phrases and clauses",
                "Hooking up two boxcars and making 'em run right",
                "Hooking up two cars to one when you say something like this choice",
                "Hooking up phrases and clauses that balance",
                "Hooking up words and phrases and clauses in complex sentences",
                "Hooking up cars and making 'em function"];

            console.log(`Conjunction Junction, what's your function? ${randomLyrics(songLyrics)}`);
